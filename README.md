## Описание проекта
Стартап продает продукты питания. Необходимо посмотреть, как пользователи мобильного приложения реагируют на изменение шрифта.

__Цель исследования:__
1. Изучить воронку событий.
2. Изучить результаты A/А-эксперимента.
3. Изучить результаты A/В-эксперимента.
4. По результатам сформулирую вывод об эффективности нововедений. 

__Ход исследования:__

В файле `/datasets/logs_exp.csv`, представлена информация об контрольных и экспериментальной группе. 
Перед анализом проверю данные на наличие пропущенных значений, дубликатов, типов данных в столбцах. Затем я проанализирую воронку событий и последовательность действий пользователей в мобильном приложении. Изучив результаты экспериментов, я сделаю вывод о том, насколько эффективно использование нового шрифта в приложении.

Таким образом в моем исследовании будут следующие этапы:
1. Обзор данных.
2. Предобработка данных.
3. Проверка данных.
4. Изучение воронки событий.
5. Изучение порядка взаимодействия с приложением.
6. Изучение результатов А/А/В экспериментов.
7. Вывод об эффективности нового шрифта.
## Используемые библиотеки
*pandas, os, datetime, plotly, seaborn, stats*
